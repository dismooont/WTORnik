# ВТОРник — GitHub Pages

## Как выложить на GitHub Pages
1. Создай новый репозиторий на GitHub.
2. Залей в корень репозитория все файлы из этой папки:
   - `index.html`
   - `.nojekyll`
   - папку `tfjs_model/`
3. Открой `Settings` → `Pages`.
4. В `Build and deployment` выбери:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
5. Сохрани настройки и дождись публикации.
6. Сайт будет доступен по адресу:
   `https://USERNAME.github.io/REPO_NAME/`

## Важно
- Для живой камеры сайт нужно открывать именно по HTTPS.
- Если камера на телефоне всё равно капризничает, используй кнопку **«Сделать фото»** — она работает стабильнее.
